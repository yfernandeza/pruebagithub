# 🎓 Taller de Comandos Git y GitHub

Repositorio educativo para aprender Git y GitHub desde lo básico hasta lo avanzado.

## 📚 Contenido del Taller

Este repositorio está diseñado para un taller progresivo que cubre:

### Fase 1: Configuración Inicial
- Instalación y configuración de Git
- Estructura básica del proyecto

### Fase 2: Operaciones Básicas
- `git init` - Inicializar repositorio
- `git add` - Preparar cambios
- `git commit` - Confirmar cambios
- `git push/pull` - Sincronizar con GitHub
- `git status/log` - Ver historial

### Fase 3: Trabajo con Ramas
- `git branch` - Crear ramas
- `git checkout/switch` - Cambiar ramas
- `git merge` - Fusionar ramas
- Resolución de conflictos

### Fase 4: Pull Requests
- Crear PRs en GitHub
- Revisar código
- Comentarios y sugerencias
- Merge desde GitHub

### Fase 5: Issues y Gestión
- Crear y gestionar issues
- Etiquetar issues
- Vincular commits a issues

### Fase 6: Tags y Releases
- Crear tags
- Versionar código
- Crear releases

### Fase 7: Revertir Cambios
- `git revert` - Deshacer commits
- `git reset` - Resetear estado
- `git restore` - Restaurar archivos

### Fase 8: Stash y Almacenamiento
- `git stash` - Guardar cambios temporales
- Aplicar y eliminar stashes

### Fase 9: Historial y Búsqueda
- `git log` avanzado
- `git grep` - Búsqueda en código
- `git blame` - Rastrear cambios

### Fase 10: Colaboración
- Fork y clone
- Upstream remotes
- Sincronizar forks
- Contribuir a proyectos open source

## 🚀 Cómo Usar Este Repositorio

1. **Clone el repositorio**
   ```bash
   git clone https://github.com/yfernandeza/pruebagithub.git
   cd pruebagithub
   ```

2. **Siga las fases en orden**
   - Lea la documentación en `/documentacion`
   - Ejecute los ejemplos en `/ejemplos`
   - Practique con el repositorio

3. **Realice commits propios**
   - Cree ramas para experimentar
   - Haga commits bien formateados
   - Cree pull requests

## 📖 Materiales

- **Ejemplos**: Revise `/ejemplos` para ver código en acción
- **Scripts**: Ejecute scripts en `/ejemplos/scripts`
